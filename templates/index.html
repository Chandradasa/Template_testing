Responsive web design

Udacity :Responsive Web Design Fundamentals Mobile market getting bigger, hence web page needs to be responsive to different devices

Viewport

Device Independent Pixels: browser will see DIP, then gets scaled to hardware pixels
If there is no meta viewport, browser will think 980px DIP when it has only 360 dips. The browser with font-boosting which tries to enlarge some content. A mess.

Meta name ="viewpoint" content="width =device-width,initial-scale=1"

w3school:viewpoint Use relative widths not absolute eg % not px, to ensure image is scaled to each device.
